【Mac】打造vs code最强编辑器

什么是vs code

关于什么是vs code，我们先来看看官网介绍

Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.js and has a rich ecosystem of extensions for other languages and runtimes (such as C++, C#, Java, Python, PHP, Go, .NET).

Visual Studio Code是一个轻量级但功能强大的源代码编辑器，可在桌面上运行，适用于Windows、macOS和Linux。它内置了对JavaScript、TypeScript和Node.js的支持，并具有丰富的其他语言和运行时扩展生态系统（如C++、C#、Java、Python、PHP、Go、.NET）

vs code全称Visual Studio Code，轻量但强大的代码编辑器，能在多系统多平台桌面化使用，支持丰富的编程语言和扩展生态。

vs code的优点

  • 免费开源。免费开源永远放在第一位，谁不喜欢白嫖
  • 轻量化。不需要大量下载，内存占用少。
  • 丰富的插件。通过安装各种插件来打造自己喜欢的编辑器风格和运行多种编程语言。

就这三点，就足以让你选择它，作为计算机人员，需要学习很多编程语言，像shell、html、sql、c、c++、python、R等等，如果一门语言装一个编辑器，就有十几个，如果还是付费的化，那就更不用多说，而vs code横空出世，拥有不比专用编辑器的代码自动补齐、自动修复等功能，还能集所有语言于一体，只能直呼666。

vs code的缺点

我个人认为，vs code的最大的缺点就是有时可以正常运行，有时就不行。有时总会出现让人费解的bug，有时又能很好的运行，举两个例子吧。

  1. 在vs code运行 python文件时，会出现

    SyntaxError: Non-ASCII character ‘xef’ in file

    原因是，代码中有中文注释，而python默认编码格式是UTF-8，需要在代码开头加上

    #\ -*- coding: utf-8\ -*

    不过到后面，会发现不加上这一句它又可以运行。

    在python专用编辑器pycharm就不会出现这种情况。

  2. 还有就是路径问题，昨天做爬虫scrapy，明明在pycharm中根目录函数调用子目录爬虫是完全没问题的，但是放到vs code中就会这样报错

    # no active project Unknown command: crawl

当然还有环境变量的配置等等，还是有使用门槛的，不过幸运的是，Mac用户相对就还好，主要是win的配置有些复杂。

所以vs code也是有缺点的，肯定是没有专用编辑器那么好用，世上并没有十全十美的东西。

初识vs code

第一遇到vs code是在大学开学，非常幸运，认识一个直系学长（姓于，我称老于，对我刚步入大学倍加关照，万分感谢），刚开学就告诉我们要学c语言，但是用的编辑器是Dev-c++，还有一个什么编辑器忘了，反正更加**，所以他主动提出帮助我们安装vs code和python，不过我的还好，因为Mac从官网下vs code一路无脑next，然后打开即使用，装一些插件就好了，同行的几位win用户要配置环境变量就相对麻烦。

不过我并没有使用多长时间，因为我后来使用了Xcode，MacOS系统的专属代码编辑器，界面是体验感是真的nice，但是那个包12个G是真的大，而且支持的语言也比较少，还有就是代码更新速度慢、运行速度慢，时不时出点小bug。但是谁让我是果粉呢微笑^_^。

后面我又把vs code删了，因为莫名其妙的bug让我无法难受，当时学习c++的时候，我决定拿出来使用，但是不知道为什么就突然都运行不了，显示的好像是配置问题。我非常气愤，就一口气直接把它卸载了，用我的学生优惠开始一年免费使用权的clion，也就是说我走的是一门语言一个编辑器的路。

我是昨天把它下回来的，因为突然看见室友在用vs code，再看见我一电脑的编辑器，我就决定开始折腾一下了，把它打造成最强编辑器，卸掉其他专用编辑器。

废话不多说，直接开始吧。

vs code安装

因为我是Mac，所以我的安装都是针对Mac的。

安装很简单，无脑安装，官网下载地址

选择适合自己电脑的版本.zip，据说intel和M芯片都分开了，注意安装版本。

安装后，一般在访达的下载里面，将它拖到应用程序，就可以开始像软件一样打开使用了。

开始界面都是英文的，中国人当然使用中国字，我们在插件商店去下载汉化包。

最左边一栏，从上至下依次是资源管理器（你的文件啥的都在这里），搜索，运行和调试，然后是扩展，打开扩展搜索Chinese点击安装重新启动即可全局设置。

vscode的界面很友好，你把鼠标放在对应的按钮上，它会显示对应的功能名称和快捷键。

只要使用过编辑器，一些基本功能使用都是没问题。
